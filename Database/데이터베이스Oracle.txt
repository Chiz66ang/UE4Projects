CONNECT hr/hr@LOCALHOST:1521/XEPDB1;
//hr계정으로 로컬호스트 1521포트에 있는 XEPDB1 데이터베이스에 연결하겠다.
//만약이게 안된다면 CONNECT SYS(최고 관리자)로 다시 접속한 후에 시도해본다.
//hr 계정은 데이터베이스 연습으로 가장 많이 쓰는 계정으로 데이터베이스 안에 정보 입력, 수정, 삭제가 모두 가능한다.

//여기까지하면 hr계정 세팅이 모두 완료가 된 것이고, 이후에 접속 할 때는 다시 설정해주지 않아도 된다. (sql developer에서도 계정만 추가해주면 된다)

SELECT 문장 (데이터베이스에서 특정 행, 열을 선택하여 검색한다)
SELECT A FROM B;
B 엔티티에서 (B에 속해있는) A 열을 가지고 와라.
A 자리에 *이 들어갈 경우 모든 열을 가지고 온다.
한개 이상의 열을 가지고 오려면 SELECT A,C FROM B;
한개 이상의 열에서 특정 행을 가지고 오려면 SELECT A,C FROM B WHERE A='행정보';
즉, WHERE이 붙으면 행을 가지고 온다 (행조건).
SELECT last_name, salary FROM employees WHERE last_name='Austin';
'홑따옴표'는 날짜, 문자열을 표현하는 기호이다.

로그인 후 쓸 수 있는 명령어
SELECT tname FROM TAB;
//TAB에 있는 tname 열을 가져와라.
