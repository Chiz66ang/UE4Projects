게임 UI/UX 프로그래밍

- UI (User Interface): 사용자 인터페이스를 비주얼 측면, 레이아웃 측면에서 고려하는 것.
- UX(User Experiecne): 사용자 인터페이스를 사용할 때 경험하는 느낌, 펀의성, 직관성을 고려하는 것
- 게임을 실행하면 화면 중앙에 버튼이 하나 있고,
그 버튼을 누르면 3인칭 프로젝트가 실행된다.

게임 서버 프로그래밍
- 게임 플레이어간의 채팅
- 다중 이용자 게임
- Replication

아이템 박스/ 인벤토리 만들기

Custom Widget
- 여러번 사용해야하는 위젯은 한개를 생성하여 재활용한다.
- 컬러 버튼 만들기
- 아이콘 (32*32) 5개 준비
- 이미지 버튼 만들기 (이미지의 크기: 32*32)

InventoryBox 위젯 만들기
- Verticle Box
- Text
- Uniform Grid Panel

- 키보드에서 i 키를 누를 때마다 InventoryBox 슬라이딩

3인칭 캐릭터가 이동하면서 바닥에 놓인 아이템을 충돌한다.
충돌한 아이템의 이미지(Texture2D)가 아이템 인벤토리 슬롯으로 복사된다.

-BP_PickupActor (이미지 위젯, 텍스트 위젯을 포함)
-BP_PickupActor는 항상 카메라를 정면으로 바라보도록 FindLookatRotation 이용
-BP_PickupActor에서 충돌감지 > 충돌한 캐릭터의 참조를 구함
-캐릭터의 함수호출 > 함수에서는 인벤토리 박스의 슬롯에 이미지 전달.
